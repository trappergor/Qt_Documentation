Устаревшие члены для QTextCodec

Следующие члены класса QTextCodec устарели. Они предназначены для поддержания работоспособности старого исходного кода. Мы настоятельно не рекомендуем использовать их в новом коде.

Статические открытые члены

(obsolete) QTextCodec *codecForTr()

Описание функций-членов

[static] QTextCodec *QTextCodec::codecForTr()

Возвращает кодек, используемый QObject::tr() в своем аргументе. Если эта функция возвращает 0 (по умолчанию), tr() предполагает Latin-1.
