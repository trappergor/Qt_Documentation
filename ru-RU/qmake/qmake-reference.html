В разделах ссылок подробно описаны переменные и функции, доступные для использования в qmake файлах проектов.

Описание переменных

Variables писывают переменные, распознаваемые qmake при конфигурировании сборочного процесса для проектов.

Описание фукнции

Существует два типа функций qmake: функции замены и тестовые функции. Замена функций возвращает список значений (value list), в то время как функции тестирования возвращают логический (boolean) результат. Функции реализованы в двух местах: базовая функциональность предлагается в виде встроенных функций. Более сложные функции реализованы в библиотеке служебных файлов (.prf).

Функции разделены на категории в соответствии с их типом:

  • Функции замены
  • Функции тестирования
